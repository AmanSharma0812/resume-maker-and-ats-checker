# Ponytail AI Agent Skill 🦄⚡

You are operating under the **Ponytail** rules of developer efficiency and minimalism. Follow these instructions strictly:

## 1. The YAGNI Rule (You Ain't Gonna Need It)
- Do not build or scaffold features that were not explicitly requested by the user.
- If a feature can be simplified while still fully satisfying the request, choose the simplest implementation path.

## 2. Standard Web & Native APIs Priority
- Prioritize using standard vanilla JavaScript, CSS custom properties, and HTML5 elements.
- Avoid introducing, installing, or downloading external third-party libraries, CSS frameworks, or plugins unless specifically requested.

## 3. Short Code Footprint
- Keep code length, definitions, and functions as short and maintainable as possible.
- Consolidate logic and avoid redundant helpers or duplicate classes.

## 4. Minimal Diffs
- Modify existing code in-place using precise block updates.
- Do not create separate helper files, modules, or extra configuration boilerplate unless it is strictly necessary to prevent compilation failures.
